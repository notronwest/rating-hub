Build notes
• Section placement: between WMPC Analysis and Review Queue, so the coach progression goes patterns → stats → flagged-moments.
• Recommendations persist on the existing analysis_topic_recommendations table with topic_id = "stat.kitchen_arrival_serving" — same FPTM/drills/overall-note schema as a WMPC topic, no new editor needed.
• Add tracking goes in a new coach_stat_reviews(analysis_id, player_id, stat_key) table — just a presence flag.
• Per-rally data lives in PB Vision's augmented insights (kitchen_arrivals per player per rally). To unlock individual-instance review on this stat we need to import that field into our DB; the import is one new column on rallies + a small change to the augmented-insight import path.
• Seek: each tile starts the clip at the beginning of the rally.

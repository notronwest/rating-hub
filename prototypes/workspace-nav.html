Nav chain from this bar

From any game view, the user can:

  • ← session chip → back to SessionDetailPage
  • ⏴⏵ arrows → jump to prev/next game in the same session without bouncing
  • tab row → switch mode (Overview / Analyze / Review) in place
  • ✕ on drill chip (Review only) → back to review overview for this game
  • status chip → passive, but shows "Review Started" / "not started" / "• N sequences · M flags"
